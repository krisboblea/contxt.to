## API Contract (MUST NOT CHANGE)
The UI depends on these exact interfaces:

### Server Action: `createContext(data)`
```typescript
// Input: { title: string, summary: string, content: string, tags?: string }
// Success: { success: true, slug: string, claimToken: string, url: string }
// Error:   { success: false, error: string }
```

### API: `POST /api/contexts`
```
// Body: { title, summary, content, tags? }
// 201: { slug, url, claimToken? }
// 400: { error: string }
```

### API: `POST /api/claim`
```
// Body: { slug, token, email }
// 200: { claimed: boolean }
// 400: { error: string }
```

### API: `POST /api/generate-meta`
```
// Body: { content: string }
// 200: { title: string, summary: string, source: "extraction" }
```
